# conda environments manage

## create conda new environments
```
$ conda create --name py36 python=3.6 pip numpy
```
## conda env list
```
$ conda env list
```
## activate py36 env
```
$ conda activate py36
```
> you will see: (py36)$  in terminal
```
$ python 
```
> into conda python 3.6 env
```
$ exit()  
```
>exit

## check package explicit link
```
$ conda list --explicit
```
